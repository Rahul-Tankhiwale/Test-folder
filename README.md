## 📌 Project Description

**Expense Tracker** is a full-stack web application that helps users manage and track their daily expenses efficiently.
Users can securely log in, add expenses, view their financial data, and manage transactions in a simple and responsive interface.

The application uses **JSON Web Tokens (JWT)** for authentication and stores the token in **localStorage** to maintain user sessions securely across the app.

The frontend is deployed on **Vercel**, and the backend API is hosted on **Render**, making the application production-ready and accessible online.

## 🔐 Authentication Flow

1. User logs in with credentials.
2. Server generates a **JWT token**.
3. Token is stored in **localStorage**.
4. Axios interceptor attaches the token to API requests.
5. Protected routes verify the token before allowing access.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Rahul-Tankhiwale/Test-folder.git
cd expense-tracker
```

### 2️⃣ Install Dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

### 3️⃣ Run the Application

Start Backend:

```bash
node server.js
```

Start Frontend:

```bash
npm run dev
```
